Рекомендации по рабочему процессу
	•	Перед началом работы:
1. Всегда выполняй команду git pull, чтобы получить последние изменения

	•	После внесения изменений:
	Добавь файлы: git add ........

2.	Создай коммит:

    git commit -m "Описание изменений"

3.	Отправь изменения на GitHub:

    git push



Перезагрузка сервера

uvicorn my_project.app.main:app --reload
